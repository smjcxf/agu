## 一、本次修改概览（6月16-17日凌晨）

### 1. 金股池入池门槛
- **原来**：signal_count >= 1 就入池
- **现在**：signal_count >= 2 才入池
- 单信号股票全部过滤

### 2. 研报股过期
- **原来**：投行研报股永久保留
- **现在**：所有股票统一20个交易日过期（GOLD_POOL_DAYS = 20）

### 3. 上涨趋势条件收紧
- 条件2：量能确认从 1.3倍 → **1.5倍**，新增 MA5 > MA10
- 条件3：成交量必须高于5日均量 **20%**（原来是均量>均量即可）

### 4. 评分公式（多因子）
| 因子 | 规则 |
|------|------|
| 基础分 | signal_count × 3 |
| 动量奖励 | 近20日涨幅 20%-50%：+1~2分 |
| 过热惩罚 | 涨幅 >50%：-1分 |
| 机构加分 | fund_type=机构：+0.5分 |
| 游资加分 | fund_type=游资：+0.3分 |
| 研报加分 | 有投行研报：+0.5分 |
| RSI风控 | RSI>70：-1分；RSI<30：+0.5分 |
| 连续共振 | 连续N天：+0.5×N（上限+3） |

### 5. 动态止损/目标
| 板块 | 止损 | 目标 |
|------|------|------|
| 主板 | -7% | +15% |
| 创业板/科创板 | -10% | +20% |
| 港股 | -8% | +15% |
| 高波动（20日>30%） | 额外+3% | — |

### 6. 涨停延迟
- 当日涨停股票标记为 WATCH，不直接推荐追板

### 7. 推荐列表统一标准
- **统一标准**：只推荐当前 signal_count >= 3 的股票（三线共振）
- 数据源合并：scan_result + watch_result + gold_pool（含历史max_signal>=3且当前仍>=3的）
- 用户只看前三只（按评分降序）

### 8. 三线历史追踪
- 06-16 只显示当前有效的3只（和林微纳、吴通控股、天承科技）
- 6/1-6/16 全部工作日日期已补全

### 9. 金股池瘦身
- 616只 → **312只**（清理单信号和0信号僵尸股）

## 三、部署流程（重要！）

**每次修改数据后必须执行：**
```bash
cd /e/workspace/stock-scanner
python update_data_v2.py --fast    # 更新dist/index.html内联数据
python deploy_now.py --force        # 部署到GitHub Pages
```

**不能只运行deploy_now.py！** update_data_v2.py负责把 data/*.json 注入到 dist/index.html 母版中。如果跳过这步，前端会显示旧数据。
