## 四、已知风险/注意事项（给阿狸咪）

### 1. 新加坡海峡时报指数（⚠️）
- `gb_sti` 不是实际STI指数，是iShares MSCI Singapore ETF（美元计价，约13元）
- 涨跌幅可能不准确，仅作参考
- 如果 -12.40% 这种极端值反复出现，可以在 `fetch_overnight_brief.py` 中移除新加坡

### 2. 北向净买额永久停更
- 港交所2024年5月起不再披露北向实时买卖数据
- akshare 数据止于 2024-08-16
- 前端已显示橙色公告说明

### 3. 抱团数据切换
- 旧 NeoData API 已死，不可回退
- 新数据源：东方财富免费端口（行业+概念资金流向）
- 数据字段与旧版不同：新的是 `industry_flow`/`concept_flow`，旧的 `current_clusters` 也保留兼容

### 4. EMA趋势计算
- 在 scanner.py 中计算，需要每只股票至少100天历史数据
- 如果某只股票无足够数据，ema_up 返回 0
- EMA趋势全局可见（不需要特殊密码）

### 5. 部署锁
- `.deploy_lock` 文件在 main 分支上
- 两台机器抢锁，只有 push 成功的能部署
- 如果部署被拒绝，等3分钟锁过期再试
- 紧急情况用 `deploy_now.py --force` 跳过审计

### 6. 坚果云同步注意
- `dist/` 目录容易产生冲突文件（`-冲突-柠萌猫咪_Win10` 后缀）
- 不要手动删除冲突文件，让 `git pull --rebase` 处理
