## 四、评分公式（全新多因子）

```python
score = signal_count * 3                    # 基础分
+ days_in_resonance >= 3: +2.0              # 连续共振奖励
+ days_in_resonance >= 2: +1.0
+ 10% < pct20 < 30%: +1.0                   # 动量适中
+ 5% < pct20 < 50%: +0.5
+ turnover_rate > 5%: +1.0                  # 换手活跃
+ turnover_rate > 3%: +0.5
+ fund_type == "机构": +0.5                  # 机构资金
+ fund_type == "游资": +0.3
+ has_report: +1.0                           # 研报覆盖
- pct20 > 50%: -3.0                         # 过热惩罚
- pct20 > 30%: -1.0
- pct20 < -20%: -2.0
- RSI > 70: -1.0                            # RSI风控
+ RSI < 30: +0.5                           # 超卖加分
```

**前端兼容：** 旧数据无 signal_score 字段时，自动按原公式计算（base + chg*0.5 + pct20*0.2）。

## 六、涨停延迟机制

```python
涨停阈值 = 20.0 if 板块 in (创业板, 科创板) else 10.0
涨停延迟 = abs(当日涨跌幅) >= 涨停阈值
action = "WATCH" if 涨停延迟 else "BUY"
```

### 1. 缠论向量化优化（未修改）
- 缠论指标涉及前后依赖和状态传播，完全向量化风险太高
